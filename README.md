# Languages
Language list with language code, name and native name.

# Formats
* JSON
* XML
* PHP Array

# Install

### Install using composer
```
composer require public-data/languages
```

### Install using npm
```
npm install languagelist
```

### Install using bower
```
bower install languagelist
```
